 4.1 Prerequisites
        Node.js (v16+)
        NPM (v8+)

    4.2 Installation
        Clone the Repository:
        git clone <repository-url>
        cd backend-node
        Install Dependencies:
        npm install

    4.3 Running the Application
        Development Server:
        npm start
        Access the Application: Open your browser and navigate to <http://localhost:3000>.
